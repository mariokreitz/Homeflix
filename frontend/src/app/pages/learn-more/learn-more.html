Mehr über Homeflix erfahren

Homeflix ist deine persönliche Streaming-Plattform für zu Hause. Streame deine eigenen Filme, Serien und mehr auf all deinen Geräten - einfach, schnell und sicher.

Hauptfunktionen

Medien-Management

Verwalte deine Filme, Serien und Videos in einer übersichtlichen Bibliothek mit automatischen Metadaten.

Multi-Device Streaming

Streame auf allen deinen Geräten - vom Smartphone bis zum Smart TV, alles in deinem Heimnetzwerk.

Sicherheit & Privatsphäre

Deine Daten bleiben bei dir. Keine Werbung, kein Tracking, keine Datensammlung.

So funktioniert's

  1. 1

    Installation

    Installiere Homeflix auf deinem Server oder NAS mit Docker.

  2. 2

    Medien hinzufügen

    Füge deine Filme und Serien hinzu. Homeflix organisiert sie automatisch.

  3. 3

    Streamen

    Greife von jedem Gerät in deinem Netzwerk auf deine Medien zu.

Technische Details

System-Anforderungen

  • Docker-fähiges System (Linux, macOS, Windows)
  • Mindestens 2GB RAM
  • Ausreichend Speicherplatz für deine Medien
  • Netzwerkverbindung für Streaming

Technologie-Stack

  • Node.js & Express Backend
  • PostgreSQL Datenbank
  • Angular Frontend
  • Docker für einfache Bereitstellung
